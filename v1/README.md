# Версия робота 1 - горизонтальный спиннер

---

### Описание

Первая версия робота построена по схеме с горизонтальным спиннером. Вращающийся диск  расположен горизонтально и наносит удары по корпусу противника за счёт кинетической энергии, накопленной при вращении. Момент кручения передается через ремень

Питание осуществляется от аккумулятора 4S (14,8 В), что обеспечивает достаточную мощность для работы двигателя оружия и приводных моторов.
